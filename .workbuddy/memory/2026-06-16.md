## 添加 TTML (Timed Text Markup Language) 歌词支持

### 用户需求
用户要求添加 TTML 歌词支持。TTML 是 W3C 标准的时间文本格式，常用于 Apple Music、Netflix 等平台的卡拉 OK 歌词。

### 实现内容

#### 1. 添加 TTML 解析器
文件：`CatClawMusic.Core/Services/LyricsService.cs`

新增方法：
- `ParseTtml(string ttmlContent)` - 解析 TTML 格式字符串
- `ParseTtmlTimestamp(string timestamp)` - 解析 TTML 时间戳（支持多种格式）
- `ParseTtmlParagraph(XElement, XNamespace, TimeSpan)` - 解析 TTML 段落，提取逐字时间戳
- `ParseTtmlFromFile(string filePath)` - 从文件解析 TTML
- `ParseTtmlFromFileAsync(string filePath)` - 异步从文件解析 TTML

### TTML 格式支持

**支持的时间戳格式**：
- `HH:MM:SS.mmm`（如 `00:03:45.500`）
- `MM:SS.mmm`（如 `03:45.500`）
- 秒数（如 `225.5` 或 `225.5s`）
- ISO 8601 持续时间（如 `PT225.5S`）

**支持的结构**：
- `<p begin="..." end="...">` - 歌词行
- `<span begin="..." end="...">` - 逐字时间戳（用于卡拉 OK 效果）
- 命名空间：`http://www.w3.org/ns/ttml`（自动检测）

### 文件扩展名支持
- `.lrc` - LRC 格式（原有）
- `.ttml` - TTML 格式（新增）
- `.xml` - 可能是 TTML（新增，会自动检测是否包含 `<tt` 或 TTML 命名空间）
