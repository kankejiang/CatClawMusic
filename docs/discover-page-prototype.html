参考的设计原则（来自 IMA 知识库）

① 玻璃态风格：卡片用半透明 + backdrop-blur + 1px 描边，叠加 13% 白色高光层，营造层次而不抢内容。
② 多层次空间感（菜心设计铺《卡片设计成这样能落地吗》）：Hero 分背景层（渐变）/中间层（光晕）/内容层（文字+按钮）三层，避免"平"。
③ 一致圆角半径（三分设《构建一致的圆角半径体系》）：xs12 / sm16 / md20 / lg24 / xl26 / pill，全页统一节奏。
④ 信息节奏：每屏一个主行动（Hero 播放），横向 scroll-snap 浏览，纵向列表收敛详情，符合"发现→浏览→播放"心智。
